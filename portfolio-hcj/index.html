Tentang

Hai, Nama saya Rizky Rico Ananda Siagian , saya mahasiswa Universitas Jember Fakultas ilmu Komputer Prodi Teknologi Informasi, saya mempunyai ketertarikan pada bidang Pengembangan Web dan Desain. Kecintaan saya pada teknologi dan desain dimulai sejak saya menemukan betapa menyenangkannya mengubah ide menjadi tampilan visual yang bisa digunakan dan dinikmati orang banyak. Selama beberapa tahun terakhir, saya telah mengerjakan berbagai proyek, saya percaya bahwa kombinasi antara kode yang bersih dan estetika desain yang matang mampu memberikan pengalaman terbaik bagi pengguna.
